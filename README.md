# AI-Powered Customer Support Agent using Google ADK

Multi-tool AI customer support agent built using Google ADK with MCP integration, session memory, and sequential reasoning.

## Features
- Order status checking
- Product availability checking
- MCP tiny image previews
- Session-based memory
- Sequential multi-step reasoning
- Structured JSON responses

## Technologies Used
- Google ADK
- Gemini 2.5 Flash Lite
- MCP (Model Context Protocol)
- Python
- Kaggle Notebook

## Core Concepts Implemented
- MCP Integration
- Custom Function Tools
- InMemorySessionService
- Sequential Tool Calling
- Context Retention

## Example Queries
- "Check my order 2024"
- "Is Samsung S24 available?"
- "Check order 500 and show product image"
